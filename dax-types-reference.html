NOTE IMPORTANTE

Le mode VAL évalue l'argument immédiatement avant l'appel (eager). Le mode EXPR transmet l'expression de façon paresseuse (lazy) — elle est réévaluée dans chaque contexte de filtre à l'intérieur de la fonction. Utiliser VAL sur une mesure dans une matrice revient à la figer à une seule valeur pour toute la colonne.
